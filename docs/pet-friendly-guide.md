# 반려동물 정보 입력 가이드

> My Trip 프로젝트의 반려동물 친화 정보 입력 가이드입니다.

---

## 📋 개요

My Trip은 반려동물과 함께 여행하는 사용자들을 위해 관광지의 반려동물 동반 가능 여부 정보를 제공합니다. 사용자가 직접 정보를 제출하여 커뮤니티 기반으로 정보를 수집합니다.

---

## 🎯 주요 기능

### 1. 반려동물 정보 조회
- 관광지 상세 페이지에서 반려동물 동반 가능 여부 확인
- 관광지 목록에서 반려동물 뱃지로 빠른 확인
- 반려동물 필터를 통한 검색

### 2. 반려동물 정보 제출
- 관광지 상세 페이지에서 "반려동물 정보 제출" 버튼 클릭
- 반려동물 동반 가능 여부, 정책, 요금, 제한사항 등 입력
- 로그인한 사용자만 제출 가능

### 3. 반려동물 정보 수정
- 기존 정보가 있는 경우 "반려동물 정보 수정" 버튼으로 수정 가능
- 본인이 제출한 정보만 수정 가능 (향후 구현)

---

## 📝 정보 입력 방법

### 1. 관광지 상세 페이지 접속
관광지 상세 페이지(`/places/[contentId]`)로 이동합니다.

### 2. 반려동물 정보 섹션 확인
페이지 하단의 "반려동물 동반 안내" 섹션을 확인합니다.

### 3. 정보 제출/수정
- **정보가 없는 경우**: "반려동물 정보 제출" 버튼 클릭
- **정보가 있는 경우**: "반려동물 정보 수정" 버튼 클릭

### 4. 폼 작성
다음 정보를 입력합니다:

- **반려동물 동반 가능 여부** (필수)
  - 체크: 반려동물 동반 가능
  - 미체크: 반려동물 동반 불가

- **정책 설명** (선택)
  - 예: "소형견만 가능", "실내 반려동물 동반 가능" 등

- **추가 요금** (선택, 숙박 시설용)
  - 숫자로 입력 (원 단위)
  - 예: 5000 (5,000원)

- **크기 제한** (선택)
  - 소형견 (10kg 이하)
  - 중형견 (20kg 이하)
  - 대형견 (30kg 이하)
  - 제한없음

- **마리 수 제한** (선택)
  - 숫자로 입력
  - 예: 2 (최대 2마리)

- **추가 안내사항** (선택)
  - 자유 텍스트로 추가 정보 입력

### 5. 제출
"제출" 버튼을 클릭하여 정보를 저장합니다.

---

## ⚠️ 주의사항

1. **정확한 정보 입력**: 실제 방문 경험이나 공식 정보를 바탕으로 입력해주세요.
2. **정책 변경**: 관광지의 반려동물 정책이 변경될 수 있으므로, 최신 정보로 업데이트해주세요.
3. **개인정보**: 반려동물 정보 입력 시 개인정보는 포함하지 않습니다.

---

## 🔍 정보 확인 방법

### 관광지 목록에서 확인
- 반려동물 동반 가능한 관광지에는 🐕 뱃지가 표시됩니다.
- 반려동물 필터를 활성화하면 반려동물 동반 가능한 관광지만 표시됩니다.

### 관광지 상세 페이지에서 확인
- "반려동물 동반 안내" 섹션에서 상세 정보를 확인할 수 있습니다.
- 정책, 요금, 제한사항 등 모든 정보가 표시됩니다.
